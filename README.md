# ⚡ Mikroprosesor dan Mikrokontroler

Projek *embedded system* dan mikrokontroler menggunakan **Arduino Nano (ATmega328P)**. Repository ini berisi berbagai eksperimen *hardware* dan *software* sebagai bagian dari pembelajaran dan inovasi. 

## 📂 Daftar Projek

### 1. 📟 LCD I2C - Teks Berjalan
Projek antarmuka menampilkan teks berjalan (running text) pada LCD 16x2.
* **Komponen:** Arduino Nano, LCD 16x2, Modul I2C Backpack.
* **Komunikasi:** I2C Protocol (Pin A4 untuk SDA, Pin A5 untuk SCL).
* **Library Tambahan:** `LiquidCrystal_I2C.h` (Frank de Brabander).

### 2. 🚥 10 LED Berjalan (Knight Rider Effect)
Projek kontrol multi-output digital menggunakan konsep perulangan (*looping* / `for loop`) untuk meminimalisir penulisan kode repetitif, menghasilkan efek LED berjalan bolak-balik.
* **Komponen:** Arduino Nano, 10x LED, 10x Resistor (100 Ohm), Breadboard.
* **Pin Digital:** D2 sampai D11.
